Project Description
PetHub is a web-based pet care and product management system developed using Java, JSP, Servlets, JDBC, and MySQL. The system allows users to browse pet products, manage shopping carts and wishlists, place orders, and access pet care information. It also includes administrative functionalities for managing products and users.
